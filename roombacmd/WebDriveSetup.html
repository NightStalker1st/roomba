HowTo set up the web drive web page

  1. Connect Roomba to the your router or other linux box which is going to drive Roomba. Figure out the serial port name.
  2. Copy webcrive.html and webdrive.css to /www
  3. Copy webdrive.cgi to /www/cgi-bin. Make it executable with chmod +x /www/cgi-bin/webdrive.cgi
  4. Copy roombacmd.mpl to your roomba working directory - e.g. /roomba
  5. Edit /www/cgi-bin/webdrive.cgi and set the variables to point to your port name, baud rate, directory where roombacmd.mpl.
Send your browser to http://webdrive.html
